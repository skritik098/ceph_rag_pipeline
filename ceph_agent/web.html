Process Workflow

1

Query Classification & Planning

The user submits a query. The LLM classifies the requested operation as safe or destructive. For complex operations, it generates a high-level plan of the steps required.

2

Command Retrieval & Generation

The Retriever Agent performs a semantic search against the vector database. Using the LLM, it judges the relevance of results and builds a custom, dynamic command, complete with necessary arguments.

3

Execution & Context Storage

The Executor Agent executes the generated command on the Ceph cluster. The output from the command is captured and stored as context for the next step.

4

Analysis & Synthesis

The Analyzer Agent receives the context and analyzes the output of the execution. It synthesizes this information into a clear, final result.

5

Deliver Response

The final, synthesized answer is delivered back to the user, completing the workflow.
